# GitHub Activity Dashboard CLI

A beautiful command-line dashboard for any public GitHub user.

## Features

- 📊 View user profile, repos, followers
- 🏆 Byte-accurate language statistics
- 🔥 Contribution streak tracking
- 🌡️ GitHub-style heatmap
- 👥 Side-by-side user comparison
- 💾 Export to JSON or Markdown

## Installation

```bash
pip install -r requirements.txt
cp .env.example .env
# Add your GitHub token to .env
